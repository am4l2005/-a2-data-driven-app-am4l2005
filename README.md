# World Explorer Data App

This is a Python Tkinter data-driven application created for CodeLab II Assessment 2.

The app uses the REST Countries API to allow users to search, compare and save country information.

## Features

- Search countries by name, region, currency code or language code
- View country details such as capital, population, area, languages and currencies
- Display country flags
- Open selected countries in Google Maps
- Compare two countries side by side
- Save favourite countries during the session
- View recent search history
- Error handling for invalid searches and connection issues

## API Used

REST Countries API  
https://restcountries.com/

## Libraries Used

- requests
- Pillow
- tkinter
- webbrowser
- io
